# LAB_4_EECS348
Coding for Programming problem: #1. Football Score Possibilities &amp; #2. Temperature Conversion

#1. Football Score Possibilities:

The Football Score Possibilities program requires reading an NFL game score and determining the scoring possibilities. A score is expected to be two or higher. Thus, the program should stop if a score less than two is entered.

#2. Temperature Conversion:

Read a value representing the temperature (in Fahrenheit, Celsius, or Kelvin) and generate the following reports:

1. Convert and display the temperature based on the user’s input scale and desired target scale

2. Categorize the converted temperature into one of the following categories:

Freezing: Below 0°C (or equivalent in other scales)
Cold: 0°C to 10°C (or equivalent)
Comfortable: 10°C to 25°C (or equivalent)
Hot: 25°C to 35°C (or equivalent)
Extreme Heat: Above 35°C (or equivalent)
Provide a simple weather advisory based on the category (e.g., "Wear a jacket," "Stay indoors").
